# Quick Start

## 1. Activate the environment

```cmd
.venv\Scripts\activate
```

## 2. Review the watchlist

Edit:

```text
config/watchlist.csv
```

## 3. Run the complete test suite

```cmd
pytest
```

## 4. Execute Atlas

```cmd
python run_all.py
```

## 5. Review outputs

```text
output/latest.xlsx
output/morning_brief.md
logs/atlas.log
logs/execution_metrics.csv
data/atlas_history.db
```

- Ranking
- Summary
